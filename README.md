# Unofficial 2012 Aloha Ruby Conf Gem!

## Installation

Add this line to your application's Gemfile:

    gem 'aloha-ruby-conf'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aloha-ruby-conf

## Usage

    > require 'aloha-ruby-conf'

    > AlohaRubyConf.translate("hello")
